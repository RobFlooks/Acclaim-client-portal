Creating a New Case

1 Click "Submit Case": Navigate to the case submission form
2 Fill in debtor details:
  • Debtor name and contact information
  • Outstanding amount
  • Account number
  • Description of debt
3 Upload supporting documents: Attach relevant invoices, contracts, or correspondence
4 Submit case: Review all information and submit for processing
Accepted file formats: PDF, DOC, DOCX, TXT, JPG, JPEG, PNG, GIF, XLSX, XLS, CSV (max 10MB per file)

üí¨ Messaging System

Sending Messages

1 Navigate to Messages: Click on "Messages" in the main navigation
2 Create new message: Click "New Message" button
3 Select recipient: Choose from your recovery team or specific case handlers
4 Choose message type:
  • Case-specific: Related to a particular case
  • General: General inquiries or support
5 Write and send: Type your message and click send

Message Features

  • File attachments: Attach documents to your messages
  • Case linking: Messages automatically link to relevant cases
  • Search functionality: Find specific messages quickly
  • Response tracking: See when messages are sent and received
Communication tip: Be specific about your case when sending messages. Include account numbers or case names for faster responses.

üìÑ Document Management

Uploading Documents

1 Navigate to Documents: Click on "Documents" in the main navigation
2 Select case: Choose the case you want to upload documents for
3 Upload files: Drag and drop or click to select files
4 Add description: Provide a brief description of the document

Document Organization

Documents are automatically organized by:

  • Case: All documents grouped by case
  • Date: Chronological order of uploads
  • Type: Categories like invoices, contracts, correspondence
  • Size: File size information for management

Downloading Documents

You can download documents at any time by:

  • Clicking the download button next to any document
  • Viewing documents directly in your browser
  • Downloading multiple documents as a zip file
Security note: All documents are securely stored and encrypted. Only authorized users can access your files.

üîß Troubleshooting

Common Issues

Can't log in

  • Check your username and password
  • Ensure caps lock is off
  • Try resetting your password
  • Contact your administrator if issues persist

File upload problems

  • Check file size (max 10MB)
  • Ensure file format is supported
  • Try a different browser
  • Check your internet connection

Missing cases or data

  • Check if filters are applied
  • Ensure you're in the correct organization
  • Try refreshing the page
  • Contact support if data is missing

Performance issues

  • Close unnecessary browser tabs
  • Clear your browser cache
  • Try using a different browser
  • Check your internet connection speed

Browser Support

The portal works best with:

  • Chrome (recommended)
  • Firefox
  • Safari
  • Edge

Getting Help

1 Use the messaging system: Send a message to your support team
2 Provide details: Include screenshots and error messages
3 Check response: Monitor your messages for replies
Help tip: When reporting issues, include your browser type, operating system, and a detailed description of what you were doing when the problem occurred.
